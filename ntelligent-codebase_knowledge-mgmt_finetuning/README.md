README 2 – Intelligent Codebase and Domain Knowledge Management System
Title: Intelligent Codebase and Domain Knowledge Management System
Overview
Concept for an AI system that answers questions about a project’s codebase and documentation to help developers find information faster.
What It Does
	•	Combines code, design docs, specs, and KT documents into one knowledge source.
	•	Uses a language model plus retrieval to give grounded answers.
Tools / Methods
	•	Open-source LLMs (e.g., Llama‑2 or Mistral).
	•	Retrieval-Augmented Generation with a vector database.
