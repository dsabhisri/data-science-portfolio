README 1 – Movie Recommender System
Title: Movie Recommender System
Overview
Simple movie recommender using the small MovieLens dataset. It suggests ten movies similar in genre to a movie the user selects.
What It Does
	•	Loads movie titles and genres.
	•	Uses genre overlap to measure similarity.
	•	Returns the top 10 most similar movies.
Tools / Methods
	•	Python, pandas.
	•	Content-based filtering using genres.
